# Project Synopsis

## Blinkit Sales Analysis Dashboard (Excel)

### 1. Introduction

The **Blinkit Sales Analysis Dashboard** is a data analytics project developed using **Microsoft Excel** to analyze and visualize sales data of a simulated grocery retail company called Blinkit. The project transforms raw sales data into meaningful insights through an interactive dashboard containing charts, filters, and key performance indicators (KPIs). The dashboard helps users quickly understand sales performance, product distribution, and outlet characteristics for better decision-making. ([GitHub][1])

### 2. Objective

The main objective of this project is to analyze Blinkit's sales dataset and build an interactive dashboard that helps in identifying patterns, trends, and insights related to product sales, outlet performance, and customer ratings.

### 3. Dataset Description

The dataset used in this project includes information about grocery products and outlet performance. It contains attributes such as:

* Item Type
* Item Fat Content
* Outlet Type
* Outlet Size
* Outlet Location
* Establishment Year
* Sales Data
* Product Ratings

This structured dataset is used as the primary data source for analysis and visualization.

### 4. Project Structure

The Excel project is organized into three main sheets:

**1. Dashboard Sheet**

* Interactive dashboard displaying key metrics
* Dynamic charts and visual insights
* Filters and slicers for data exploration

**2. Database Sheet**

* Contains the raw dataset
* Serves as the data source for pivot tables and charts

**3. Sheet Design**

* Contains layout planning and formatting
* Helps maintain a visually clean dashboard interface

### 5. Tools and Technologies Used

The project was developed using the following tools and techniques:

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers and Filters
* Conditional Formatting
* Dashboard Design Techniques

The entire project is built within Excel without using any external software or plugins. ([GitHub][1])

### 6. Key Features

* Interactive Excel dashboard
* KPI metrics such as Total Sales, Average Sales, Number of Items, and Average Rating
* Visual charts including bar charts, line charts, and donut charts
* Dynamic slicers to filter data by outlet type, location, and item categories
* Clean and user-friendly dashboard layout

### 7. Insights Generated

The dashboard helps analyze various business insights such as:

* Sales distribution across outlet types and locations
* Performance of different product categories
* Impact of outlet establishment year on sales
* Comparison of product types based on fat content
* Average customer rating trends

These insights help businesses understand consumer behavior and improve operational decisions.

### 8. Conclusion

This project demonstrates how **Microsoft Excel can be used as a powerful data analysis and visualization tool**. By combining structured data, pivot tables, and interactive dashboards, the project effectively converts raw data into actionable insights. The project also showcases essential data analytics skills such as data cleaning, visualization, and dashboard creation, making it a valuable addition to a data analytics portfolio.
